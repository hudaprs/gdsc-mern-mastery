# GDSC - MERN Mastery

This is documentation on how to start your project with MERN Stack. This project used for Webinar GDSC Sriwijaya.

# Prerequisites

- node >= 18 LTS
- yarn >= 1.22.21
- mongodb 7.0.5

# Installation

1. Install Dependencies

```shell
yarn dev:install
```

2. Run Project

```shell
yarn dev:all
```
