# Controller

What you need to do here? Here's an example of CRUD Controller

```typescript
class TodoController {
  /**
   * @description Get todo list
   *
   * @param {Request} req
   * @param {Response} res
   *
   * @return {Promise<any>} Promise<any>
   */
  /**
   * @description Create new todo
   *
   * @param {Request} req
   * @param {Response} res
   *
   * @return {Promise<any>} Promise<any>
   */
  /**
   * @description Update existing todo
   *
   * @param {Request} req
   * @param {Response} res
   *
   * @return {Promise<any>} Promise<any>
   */
  /**
   * @description Delete todo
   *
   * @param {Request} req
   * @param {Response} res
   *
   * @return {Promise<any>} Promise<any>
   */
}

const todoController = new TodoController()

export { todoController }
```
